# Background Images

This folder contains alternative background images for the IceBath app.

## Current Background
- **background.png** - AI-generated ice water with frozen lake (currently active)

## Alternative Backgrounds
- **background_mountain_alt.png** - Sunny mountain landscape with snow

## To Change Background
Replace `background.png` with your preferred image, or update the filename in `drawables.xml`.

All backgrounds should be 500x500 pixels for best display on watch screens.
